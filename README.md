# Wolf Math Game - Educational Math Adventure

## Overview

Wolf Math Game is an engaging educational game designed to help young students practice and improve their mathematical skills through interactive gameplay. Players join a friendly wolf character on a mathematical adventure, solving various arithmetic problems across six different levels.

## What the Game Consists Of

The game features six unique levels, each focusing on different mathematical concepts:

1. **Addition Mission** - Addition of two numbers
2. **Subtraction Mission** - Subtraction of two numbers
3. **Three Terms Addition** - Addition of three numbers
4. **Three Terms Subtraction** - Operations with three numbers
5. **Comparison Mission** - Comparing numbers using <, >, = symbols
6. **Falling Meat Mission** - Quick calculation game where players catch falling math problems

## How to Play

### Basic Gameplay
- Navigate through the menu to select "Start" or choose specific levels
- Each level presents 3 math questions to solve
- Read the math problem and select the correct answer from the four options provided
- Earn points for each correct answer

### Special Level 6 - Falling Meat Game
- In the final level, meat items with math problems fall from the top of the screen
- Meat items appear with the following timing:
  - Meat 1: Immediately (0 seconds)
  - Meat 2: After 2 seconds
  - Meat 3: After 1 second (3 seconds total)
  - Meat 4: After 2 seconds (5 seconds total)
  - Meat 5: After 1 second (6 seconds total)
- After the initial 5 meat items appear, no additional meat appears until you select the correct answer
- Click on the numbered buttons at the bottom to select the answer that matches the falling meat item
- Try to catch as many meat items as possible by selecting the correct answers

### Controls
- Click on answer buttons to select your answer
- Use the menu to navigate between levels
- Return to the main menu at any time using the "Menu" button

## Game Mechanics

- **Progress Tracking**: The game tracks your progress through a visual progress bar
- **Scoring System**: Earn points for each correct answer
- **Level Completion**: Complete each level by answering all 3 questions correctly
- **Star Rating**: Some levels award stars based on performance
- **Timer**: Level 6 has a time limit of 60 seconds
- **Meat Catching**: In level 6, catch falling meat items with correct answers
